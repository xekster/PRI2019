Abstract

Este trabalho está dividido em três diretorias principais, sendo que cada uma contém uma parte da aplicação. A diretoria LikedSharing contém o servidor de API da aplicação, e a diretoria LinkedSharing_Interface contém o servidor de interface. Em cada uma destas duas diretorias é necessário executar o comando npm i e o comando npm start de modo a instalar os módulos necessários e a iniciar o servidor. Este pode ser acedido no browser através do link localhost:3005.

A terceria diretoria, mongo_import, não interfere no funcionamento do programa, e apenas contém a gramática utilizada para o processamento do registo de um utilizador através de um ficheiro. Este gramática serviu para gerar um arquivo .jar que é executado pelo servidor para o registo do utilizador.
