Abstract

Esta entrega consiste num pequeno servidor em node.js gerado com ajuda de Express que tem como objetivo mostrar páginas web geradas através de ficheiros pug, que em conjunto são interpretadas pelo browser de modo a criar o html da página. Estas páginas são alimentadas por modelos de dados guardados numa base de dados em Mongodb e permitem também através de diversos métodos http como POST e DELETE pedir alterações à base de dados do servidor. Nesta entrega, houve também o cuidado de oferecer uma API de dados que permite diversas operações sobre o conteúdo do servidor.
