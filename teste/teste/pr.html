Abstract

No primeiro ponto, o primeiro passo foi formatar o dataset. Isto foi feito com o comando replace do Visual Studio Code, onde foram corrigidos os erros de formatação do dataset. Depois disto, o ficheiro .json resultante foi importado para o Mongodb com o comando mongoimport -c pubs -d pubs --jsonArray jcrpubs.json. Depois de instalados os módulos do Nodejs com o comando npm i, o servidor pode ser inicializado com npm start, que coloca o servidor da API à escuta na porta 3004.

No segundo ponto, foi utilizado o Postman para responder às questões colocadas, e o output foi analisado com a ajuda do Visual Studio Code e do comando find. Depois disto, foi desenvolvida um servidor em Nodejs com o objetivo de responder aos requisitos colocados. Este utiliza o Axios para enviar pedidos à API do CLAV, e mostra os dados obtidos numa interface fácil de navegar. Depois de instalados os módulos do Nodejs com o comando npm i, o servidor pode ser inicializado com npm start, que coloca o servidor da API à escuta na porta 5026.
